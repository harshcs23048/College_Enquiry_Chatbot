# College_Enquiry_Chatbot
Rule-Based College Enquiry Chatbot using Python

## Problem
Automated handling of student enquiries.

## AI Technique
Rule-Based Chatbot

## Representation
Pattern Matching Rules

## Tools
Python

## Features
- Answers questions about admission
- Provides fee information
- Gives course details
- Hostel and placement information

## How to Run
1. Install Python
2. Run:
   python chatbot.py
